2.2 Product Functions

FarmConnect Senegal will provide the following key functions to its users:

  • Weather Forecast Access
    Display localized 3-day weather forecasts for all 14 regions of Senegal using the OpenWeatherMap API.
  • Crop Advice Guide
    Provide detailed agricultural advice for 15+ popular Senegalese crops including planting guides, care tips, and seasonal recommendations.
  • Market Prices
    Display real-time market price information for agricultural products to help farmers make informed selling decisions.
  • Community Forum
    Enable farmers to connect, share experiences, ask questions, and participate in discussions with other community members.
  • Investor Portal
    Provide information and resources for potential investors interested in agricultural projects in Senegal.
  • Tools Marketplace
    Allow farmers to browse and access information about farming tools and resources.
  • Offline Mode Support
    Allow farmers to access essential farming tips and cached data even without an internet connection through PWA (Progressive Web App) technology.

    Provide content in French and Wolof to ensure wide accessibility.
  • User Dashboard
    Personalized dashboard for registered users showing relevant weather, crops, and recommendations based on their region.

    Administrative interface for managing users, content, and system settings.

2.3 User Classes and Characteristics

There are five main user groups for the FarmConnect Senegal web application:

  • Farmers (Primary Users)
    These are the main beneficiaries of the app. They typically have basic mobile or computer experience and use the system to access localized weather forecasts, crop-specific advice, and market prices. Since many farmers may have limited literacy or digital skills, the app must be simple, intuitive, and available in local languages such as Wolof and French.
  • Agricultural Extension Agents
    Extension agents generally have more technical knowledge and may use the app as a tool in their training activities. They often require access to more detailed agricultural content and may also demonstrate the app's features to farmers in the field.
  • Agricultural Experts
    Users with specialized agricultural knowledge who can provide expert advice and validate crop recommendations within the platform.
  • Investors
    Potential investors interested in agricultural opportunities in Senegal who use the investor portal to learn about investment opportunities and connect with farming projects.
  • Suppliers
    Agricultural input suppliers who can list their products and services in the marketplace, connecting with farmers who need tools, seeds, and other supplies.
  • Administrators
    System administrators who manage the platform, including user management, content moderation, database maintenance, and system configurations.

Additional Implemented Security Measures

  • Rate Limiting: Custom rate limiting decorator protects login endpoints (5 requests/minute) and API endpoints (30 requests/minute) to prevent brute force attacks.
  • Input Sanitization: All user inputs are sanitized to prevent injection attacks and XSS vulnerabilities.
  • Security Headers: Implementation includes:
    • X-Frame-Options: DENY (prevents clickjacking)
    • X-Content-Type-Options: nosniff
    • Strict Referrer Policy
    • Cross-Origin Opener Policy
  • Environment Variables: All sensitive data (API keys, secret keys, database credentials) stored in environment variables using python-decouple.
  • Session Security: Secure session cookies with HTTPOnly flag, configurable secure flag for HTTPS in production.

Database Requirements

  • The system uses SQLite relational database for development, with PostgreSQL support for production deployment.
  • The database stores:
    • User profiles and authentication data
    • Crop advice and agricultural tips
    • Weather data cache
    • Market price information
    • Community forum posts and comments
    • System configurations
  • The database supports multi-language content, enabling crop tips to be stored and retrieved in both French and Wolof.
  • Automated daily backups ensure data is not lost.
  • The database maintains data integrity through Django ORM with proper foreign key relationships and constraints.
